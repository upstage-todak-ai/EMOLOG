# EmoLog

## What is EmoLog?

캘린더의 일정을 보고, 감정이 필요할 것 같은 때, 감정 메모 작성을 유도하는 팝업 알림을 보냅니다.
그렇게 **쉽고 간단하게**감정 관련 메모를 작성하고 메모에 대한 주제, 감정이 나온 페이지를 통해 Log를 한눈에 볼 수 있습니다.

## Why EmoLog?

힘든 날일수록 감정을 정리하려고 하면 더 어려워집니다. 머릿속이 복잡해지고, 같은 생각이 반복되며, 악순환에 빠지게 됩니다.

필요한 일은 "일기 써봐", "머리 속을 정리해봐" 같은 조언이 아니라, 먼저 다가와서 "괜찮아?"라고 안부를 묻는 것이라고 생각했습니다.

그래서 토닥이는 사용자가 필요할 때, 부담스럽지 않은 방식으로 먼저 말을 걸어줍니다.

## Core Features

### 1. 감정 기록

빠르고 간단하게 감정기록할 수 있습니다.

### 2. 통계 및 리포트

기록이 쌓이면 패턴이 드러납니다. 

### 3. 먼저 말을 걸어주는 기능

시험이 끝난 날과 같은 특정 이벤트가 발생했을 때, 사용자에게 먼저 말을 겁니다. "시험이 끝났습니다. 기분이 어떠신가요?"

### 4. 감정 Log 

각 메모에 대한 [감정], [주제] 만 한눈에 쉽게 볼 수 있는 페이지를 통해 log를 볼 수 있습니다.

## High-level Architecture

Todak은 다음 흐름으로 동작합니다.

1. 사용자가 짧은 메모로 감정을 기록합니다.
2. 시스템이 기록을 저장하고, 시간에 따라 감정 변화와 패턴을 누적합니다. 주제와 감정을 추출해냅니다.(log에 나타낼)
3. 기록과 함께 캘린더의 일정 정보와 같은 맥락을 참고하여 현재 상태를 해석합니다.
4. 감정 배출이 필요할 때를 캘린더 일정을 통해 사용자에게 메모 유도 메세지 알림을 보냅니다.
5. log를 통해 감정을 볼 수 있고, 통계 페이지에서 수치적 통계와 리포트(자신의 감정과 주제에 대한 스토리텔링)을 볼 수 있습니다.

## 구성 화면
1. 홈화면 : 메모장 -> +버튼을 눌러 메모 추가
2. Log : 내가 작성한 메모를 토대로 자동으로 주제와 감정을 추출한 log를 보여줍니다. (주관없이 객관적 사실)
3. 통계화면 : 감정과 주제에대한 수치적 통계를 확인할 수 있으며, 기간(1주일/한달) 단위로 나 자신을 분석한 스토리텔링이 들어간 레포트를 확인할 수 있습니다
